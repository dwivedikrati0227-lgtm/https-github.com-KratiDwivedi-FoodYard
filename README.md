Food Rescue & Distribution System is a web application developed to minimize food wastage by providing a platform where restaurants, hotels, event organizers, and individuals can donate surplus food. The donated food is then made available to NGOs and people in need, ensuring efficient distribution and reducing hunger.

This project demonstrates how technology can be used to solve real-world social problems through an easy-to-use web platform.

✨ Features
🍽️ Food donation registration
👥 Donor and recipient management
📋 View available food donations
🗄️ MySQL database integration
📱 Responsive user interface
🔐 Secure data storage
⚡ Simple and easy navigation
